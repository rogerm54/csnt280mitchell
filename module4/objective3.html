Objective 4.3 - Updating data using a SQL query

Updating Records

A very important procedure when using a database as the backend for a web application is the procedure to update a record. Consider the following script that initializes a database table and populates that table with some data: 


-- init.sql
drop table if exists cars cascade;

create table cars(
   id serial,
   make text,
   model text,
   price decimal(10,2),
   primary key (id)
);
insert into cars (id, make, model, price)
   values (1,'Toyota','Corolla',18500.00);
insert into cars (id, make, model, price)
   values (2,'Toyota','Camry',23495.00);
insert into cars (id, make, model, price)
   values (3,'Toyota','Highlander',30630.00);
insert into cars (id, make, model, price)
   values (4,'Honda','Accord',24460.00);
insert into cars (id, make, model, price)
   values (5,'Honda','Civic',19730.00);
insert into cars (id, make, model, price)
   values (6,'Honda','Fit',17080.00);
insert into cars (id, make, model, price)
   values (7,'Nissan','Altima',23385.00);
insert into cars (id, make, model, price)
   values (8,'Nissan','Leaf',30875.00);
insert into cars (id, make, model, price)
   values (9,'Nissan','Maxima',33495.00);
insert into cars (id, make, model, price)
   values (10,'Chevrolet','Camaro',26900.00);
insert into cars (id, make, model, price)
   values (11,'Chevrolet','Corvette',56590.00);
insert into cars (id, make, model, price)
   values (12,'Chevrolet','Malibu',22555.00);

Updating a single record

Suppose we found out that the Toyota Corolla is supposed to have a price of $18,600. We can update that record using the following SQL command: 


update cars set price=18600 where id=1;

Let's run this update and show all the cars: 


cars_db=> update cars set price=18600 where id=1;
UPDATE 1
cars_db=> select * from cars;
 id |   make    |   model    | price   
----+-----------+------------+----------
  2 | Toyota    | Camry      | 23495.00
  3 | Toyota    | Highlander | 30630.00
  4 | Honda     | Accord     | 24460.00
  5 | Honda     | Civic      | 19730.00
  6 | Honda     | Fit        | 17080.00
  7 | Nissan    | Altima     | 23385.00
  8 | Nissan    | Leaf       | 30875.00
  9 | Nissan    | Maxima     | 33495.00
 10 | Chevrolet | Camaro     | 26900.00
 11 | Chevrolet | Corvette   | 56590.00
 12 | Chevrolet | Malibu     | 22555.00
  1 | Toyota    | Corolla    | 18600.00
(12 rows)

Note that the Toyota Corolla record shows up last, because it is the latest record to be updated. Also note that the price has been updated. The basic syntax for updating a single record would be: 


update <table_name> set <field_name1>=val1 where id=<id>;

Make sure that to enclose the value in single quotes if the field is a text field.

Updating several records

Suppose we found out that Chevrolet was increasing the price of all of their cars by 3%. This could be done by running the following update query: 


update cars set price=1.03*price where make='Chevrolet';

Here is the result of running that SQL command: 


cars_db=> update cars set price=1.03*price where make='Chevrolet';
UPDATE 3
cars_db=> select * from cars;

Updating several fields in a record

Suppose you realize that the Toyota Camry record is incorrect. This record was supposed to be for a Toyota Camry Hybrid that has a price of $32,900. Here is how to update that record:

update cars set model='Camry Hybrid', price=32900 where id=2;

Here is the result of running that update query: 


cars_db=> update cars set model='Camry Hybrid', price=32900 where id=2;
UPDATE 1
cars_db=> select * from cars;

Running an update using PHP PDO

Suppose you are using a PHP script to update a record. Assume that you have already connected to the backend database and the database connection is stored in $pdo . Assume that the price of the Honda Fit is going to be updated to $18557. Also assume that the price has been stored in a PHP variable called $price and that the id for the record is stored in $id . This is what the PHP code would look like: 


$sql = "update cars set price=:price where id=:id";
$statement = $pdo->prepare($sql);
$myarray = array();
$myarray[':price'] = $price;
$myarray[':id'] = $id;
$statement->execute($myarray);
